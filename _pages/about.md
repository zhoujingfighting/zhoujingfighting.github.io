This blog is a personal space where I document my technical journey, insights, and hands-on experiments, primarily focusing on **LLVM**, **GPU compilers**, **ROCm/HIP**, and **High-Performance Computing (HPC)**.

{: .notice--info}
**Current Research & Work Focus:** LLVM frontend/backend optimization, GPU code generation (AMDGPU), ROCm runtime stack, and high-performance parallel computing.

---

## 🛠 Technical Focus (技术领域)

- **Compiler Technology**: Deep dive into LLVM/Clang infrastructure, custom optimization passes, analysis of intermediate representation (IR), and backend code generation.
- **Heterogeneous Computing**: GPU compiler toolchains, ROCm (HIP/HSA runtime) compiler compilation flow, CUDA programming, and driver-level execution mechanisms.
- **System & Performance**: Profiling GPU kernels, optimizing compute-bound and memory-bound workloads, parallel algorithms, and Linux system-level programming.
- **Infrastructure**: Building robust and reproducible development environments using Docker, devcontainers, and custom toolchains.

- **Deep Dives into Compiler Internals**: Understanding compilation pipelines of HIP/CUDA files (e.g., detailed execution flow of `hipcc`).
- **Hands-on ROCm**: Step-by-step guides for compiling ROCm runtime libraries from source and configuring compiler toolchains.

- **System Tooling**: Practical tips on configuring remote development environments (SSH, devcontainers, build pipelines).
